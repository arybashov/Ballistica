# Ballistica

Учебный браузерный симулятор внешней баллистики в стиле PaperPlane: 3-DOF point-mass модель, численное интегрирование RK4, стандартная атмосфера ISA и табличная зависимость коэффициента сопротивления от числа Маха.

Проект предназначен для демонстрации физики полета тела в атмосфере и проверки численной модели. Это не таблицы стрельбы, не fire-control система и не инструмент для наведения, корректировки огня или расчета боевого применения.

## Быстрый запуск

Требования:

- Node.js для тестов.
- Python 3 для локального HTTP-сервера.
- Современный браузер.

Запуск локального сервера:

```powershell
python server.py
```

Сервер откроет каталог проекта по адресу:

```text
http://localhost:8000
```

Интерфейс симулятора доступен на главной странице:

```text
http://localhost:8000/
```

Техническое описание модели доступно как обычная HTML-страница:

```text
http://localhost:8000/whitepapers.html
```

Браузерная часть подключает `physics.js` как глобальный API, затем `scene.js` и `app.js`. Основные сущности доступны в глобальной области: `Projectile`, `simulateShot`, `PROJECTILE_PRESETS`, `atmosphereAt`, `cdFromMach`, `analyticVacuumImpact`.

## Тесты

Запуск:

```powershell
npm test
```

Команда выполняет `node tests/physics-validity.test.js`. Тесты загружают `physics.js` в VM-контекст Node.js и проверяют:

- стандартную атмосферу на уровне моря и около 11 км;
- рост сопротивления в трансзвуковой области `Cd(M)`;
- совпадение вакуумной траектории с аналитической параболой;
- уменьшение дальности и скорости удара при включенном сопротивлении;
- учебный 152-мм пресет на грубый диапазон дальности, высоты и времени полета;
- боковой снос при поперечном ветре;
- влияние массы при одинаковом диаметре.

## Модель

Модель в `physics.js` рассматривает снаряд как материальную точку с координатами `x, y, z` и скоростями `vx, vy, vz`. Вращение, нутация, гироскопическая устойчивость, деривация, Кориолисова сила, кривизна Земли, изменение массы и работа взрывателя не моделируются.

В конструкторе задаются длина, диаметр, центр тяжести от носа, оценочный центр давления от носа и спин. Эти параметры участвуют в упрощенной угловой модели: ось корпуса имеет собственную ориентацию и угловую скорость, плечо `ЦД - ЦТ` создает аэродинамический момент, спин добавляет демпфирование/гироскопическую стабилизацию, а угол атаки меняет нормальную силу и эффективный `Cd`.

Это все еще не полноценная артиллерийская 6-DOF модель: нет прецессии, деривации и точных моментов инерции конкретного снаряда. Но ЦТ, ЦД и спин уже меняют траекторию, а не только подписи в интерфейсе.

Сила сопротивления:

```text
D = 1/2 * rho * Vrel^2 * Cd * A
```

где `rho` - плотность воздуха, `Vrel` - скорость относительно локального ветра, `Cd` - коэффициент сопротивления, `A = pi * d^2 / 4` - площадь миделя. Ускорение сопротивления направлено против относительной скорости. Вертикальная ось дополнительно получает ускорение `-g`.

Ветер задается тремя компонентами `windX`, `windY`, `windZ`. Турбулентность задается как детерминированное гладкое поле порывов с амплитудой `turbulence` и пространственным масштабом `turbulenceScale`. Внутри RK4 не используется `Math.random()`, поэтому повторный расчет с теми же параметрами дает ту же траекторию.

Интегрирование выполняется классическим методом Рунге-Кутты 4-го порядка с шагом `dt`. При пересечении земли результат линейно интерполируется до `y = 0`.

## Атмосфера и сопротивление

Атмосфера реализована как приближение ISA до 20 км:

- 0-11 км: тропосфера с линейным спадом температуры;
- 11-20 км: изотермический слой;
- высота зажимается в диапазон 0-20000 м;
- плотность может масштабироваться параметром `densityScale`.

`Cd(M)` задается сглаженной учебной таблицей `MACH_CD_TABLE`. Она отражает общий тренд: умеренное дозвуковое сопротивление, резкий трансзвуковой рост и последующее снижение в сверхзвуковой области. Это не кривая конкретного боеприпаса и не табличные данные для практической стрельбы.

## Пресеты

В `PROJECTILE_PRESETS` есть учебные конфигурации:

- `shell152Of540Like` - приблизительный 152-мм ОФ-снаряд: масса 43.5 кг, диаметр 152.4 мм, начальная скорость 655 м/с.
- `vacuumCheck` - вакуумный контроль без сопротивления.
- `mortar120Like` - медленная 120-мм мина для проверки навесных траекторий.

Параметры подобраны для демонстрации и тестирования численной модели. Они не должны восприниматься как паспортные или эксплуатационные данные вооружения.

## Источники и верификация

Верификация в проекте строится на нескольких независимых опорах:

- аналитическое решение вакуумной параболы без сопротивления;
- эталонные значения стандартной атмосферы на уровне моря и около 11 км;
- качественно ожидаемый профиль `Cd(M)` с трансзвуковым ростом сопротивления;
- инварианты поведения: сопротивление уменьшает дальность, 3D-ветер меняет дальность/высоту/боковой снос, турбулентность детерминированно меняет траекторию, большая секционная плотность лучше сохраняет скорость.

Локальная книга `Doc/габица_152.djvu` полезна как справочный историко-технический контекст по 152-мм системе и терминологии. Текущая модель не извлекает из нее таблицы автоматически, не копирует таблицы стрельбы и не претендует на совпадение с ними. Любое сравнение с книгой должно оформляться как отдельная исследовательская проверка с явным указанием страницы, режима стрельбы, начальных условий и допущений.

## Ограничения и безопасность

Ключевые ограничения:

- 3-DOF point-mass вместо 6-DOF жесткого тела;
- спин, ЦТ и центр давления учитываются упрощенно; это не заменяет полноценную 6-DOF модель;
- нет вращения Земли, кривизны Земли и полной метеосводки по слоям;
- базовый ветер задается тремя постоянными компонентами, турбулентность является учебным гладким полем порывов;
- `Cd(M)` учебный, не калиброван под конкретный снаряд;
- нет рассеивания, износа ствола, температуры заряда, баллистического коэффициента из реальных таблиц;
- нет fire-control логики, обратного решения по цели и поправок для боевого применения.

Используйте проект только для обучения, визуализации и инженерной проверки численной схемы.
